What is Python?

Python is a high-level, versatile programming language known for its readability and simplicity. It’s widely used in various fields such as web development, data science, artificial intelligence, and automation. Python is beginner-friendly and has an extensive standard library that supports many programming tasks.

Key features

  • Easy-to-read syntax
  • Supports multiple programming paradigms (e.g., procedural, object-oriented)
  • Cross-platform compatibility
  • Large community and support resources

Installation Guide

  1. Download Python: Visit the official Python website and download the latest version for your operating system.
  2. Install Python: Run the installer and ensure you check the “Add Python to PATH” option.
  3. Verify Installation: Open a terminal or command prompt and type: 
    python --version
    This command should output the version of Python you installed.

Python Syntax Basics

Python syntax is simple and emphasizes readability. Here are a few key points about Python syntax:

  • Comments: Use the # symbol for single-line comments 
    # This is a comment
print("Hello, World!")  # This line prints a message
  • Indentation: Indentation (usually 4 spaces) defines code blocks, unlike other languages that use braces {}. 
    if 5 > 2:
    print("Five is greater than two!")  # Indented block
  • Printing Output: Use the print() function to output data to the screen. 
    print("Hello, World!")

Variables and Data Types

Variables

Variables are used to store data.In Python, variables are created by assigning a value using the = operator. Python supports various data types like string,Integer,float and boolean.

Example Data Type Description
age = 25 Integer Whole numbers
height = 5.8 Float Decimal numbers
name = "Alice" String Text data
is_student = True Boolean True or False values
result = None None Represents no value or empty state

Operators in Python

Python supports various operators for performing calculations and comparisons.

  • Arithmetic Operators: Used for basic mathematical operations.
    x = 10
y = 3
print(x + y)    # Addition: 13
print(x - y)    # Subtraction: 7
print(x * y)    # Multiplication: 30
print(x / y)    # Division: 3.333...
print(x % y)    # Modulus: 1
  • Comparison Operators: Used to compare values, returning True or False.
    print(x == y)   # Equal to: False
print(x != y)   # Not equal to: True
print(x > y)    # Greater than: True
print(x < y)    # Less than: False
  • Logical Operators: Used to combine conditions.
    a = True
b = False
print(a and b)  # Logical AND: False
print(a or b)   # Logical OR: True

Control Flow

Control flow allows you to execute certain parts of code based on conditions or loops.

Conditional Statements

  • If Statements: Use if, elif, and else to execute code based on conditions. 
    age = 18
if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")

Loops

  • For Loop: Used to iterate over a sequence.
    for i in range(5):
    print(i)  # Outputs: 0, 1, 2, 3, 4
  • While Loop: Repeats as long as a condition is True.
    count = 0
while count < 5:
    print(count)
    count += 1  # Increment count by 1

Functions

Functions help organize code into reusable blocks. Define a function using the def keyword.

def greet(name):
    return f"Hello, {name}!"

print(greet("Alice"))  # Outputs: Hello, Alice!

Functions can take multiple arguments and return a value.

def add(a, b):
    return a + b

print(add(5, 3))  # Outputs: 8

Data Structures

Python has several data structures to organize and manage collections of data.

List

An ordered, mutable collection of items.

fruits = ["apple", "banana", "cherry"]
fruits.append("orange")   # Adds "orange" to the list
print(fruits)  # Outputs: ['apple', 'banana', 'cherry', 'orange']

Dictionaries

Stores data as key-value pairs.

person = {"name": "Alice", "age": 25}
print(person["name"])     # Outputs: Alice

Tuples

An ordered, immutable collection.

coordinates = (10, 20)
print(coordinates[0])     # Outputs: 10

Sets

An unordered collection of unique items.

unique_numbers = {1, 2, 3, 3}
print(unique_numbers)     # Outputs: {1, 2, 3}

Working with Modules

Modules are Python files containing functions and variables. You can import built-in or custom modules.

  • Importing a Built-in Module:
    • import math
print(math.sqrt(16))  # Outputs: 4.0
  • Creating a Custom Module:
    1. Create a file my_module.py with the following code. 
      def add(a, b):
    return a + b
    2. Import and use it in another file.
      from my_module import add
print(add(3, 4))  # Outputs: 7
